# Browser Benchmark Suite

A lightweight toolkit for automated browser performance testing.
Runs headless Chromium benchmarks and exposes results via REST API.

## Features
- Automated page load timing
- JavaScript execution benchmarks
- Concurrent session testing
- REST API for remote triggering
